Querying the text

The idea is to provide to the user an overview of the text exploiting three different ways of analysis provided by the main useful digital tools. How showed in the section about the encoding phase, the text has been marked in a way that could made possible some actions on it. The first step was realized with some queries done with both XPath (for the extraction of the data) and Python (for the refining and the structuration of the data collected); the results obtained by the queries will be then necessary to the following step of analysis.

The Queries

The first datum required is the collection of all the "key" attributes related to the <persName> tag, this action enable the achievement of the list of all the characters named in the story. Since the result of this query returns a list containing every key in all its occurencies, it's necessary to refine this structure in order to visualize a list with just one occurency per character. In this case, the possibilities are many; personally, I choosed the most suitable to my abilities. Oxygen makes a user able to embed the results of his query in different formats, I decided to export in a .txt format the collection obtained and then to work on it with Python, where I sorted the list and then created a set.

XPath

//persName/@key

Once the list of character has been collected, the second step is to obtain the different names under which every character appears. This information is necessary for the following step, that about the computation of the occurencies of every character trough Voyant.

Xpath

//persName[@key='WilliamLucas']//text()

How this data can help our analysis

In the section about the textual analysis executed with Voyant, the data collected will be reused to obtain the occurencies of every character in the txt, in order to establish a sort of gerarchy of the importance of every character inside the story. Many other informations could be reached quering the text with Xpath; in this case, it has been decided to extract just the information needed to proceed with the analysis.
