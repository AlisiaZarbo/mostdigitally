Some Analysis with Voyant

The codification and the collection, trough the Xpath queries, were both useful to the following work of statistycal analysis on the text. With the list of character and the different names under which they appear in the text, it is now possible to obtain the number of occurrencies of every character inside the text, in order to establish the importance of every name inside the story. A second different work of analysis, done with Voyant, is wider and centered on the text all, providing the main words used inside the text.

Cases of homonomy and other issues

In some cases, to measure the frequency of a name inside the text, it wasn't sufficient just the computation of the of the word's occurencies; since some characters "share" the same given name, searching for the frequency of that word could compromise the analysis providing not trustable results. In this case, it can be useful to use another tool provided by Voyant, that one which show the correlation between two words. Even if this tool could be suitable to a similar research, it requires a work of surpevision and a lot of time. For this reason, it has been decided to exploit that "secondary" terms that could help to qualify the context in relation with the character about which the program is searching. This obviously implies the lost of some occurrence in the text, the exact number of appearances for every character inside the text cannot be computed using the tools offered by Voyant.

Catherine Bennet | Catherine de Bourgh

In this case, the characters analysed share the same name; the best way to proceed, how explained in the previous section, is to exploit some "add names" that confirm the association of the word to that specific character.

Mr Wickham | Mrs Wickham

In this second case, the characters share the same surname; since the author choice to use the surnames and the appellations to define the characters, it possible to take advantage of this peculiarity using them for the analysis.

Conclusions

Proceeding in this way, is is possible to obtain quite accurate results but not so exacts. In this case, this data were needed just to have an idea of the weight of every character inside the story, and so they can be considered sufficients, usefuls. But in other cases, it is better to proceed with other tools of textual analysis.
